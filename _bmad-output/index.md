## Getting Started

### Prerequisites
- Node.js 20+
- npm 10+
- Supabase account (optional)

### Quick Start

```bash
# Install dependencies
npm install
cd web && npm install && cd ..

# Set up environment
cp .env.example .env

# Run tests
npm test

# Start API (Terminal 1)
npm run api

# Start Web (Terminal 2)
npm run web
```

### URLs
- **API:** http://localhost:3010
- **Web:** http://localhost:3000
- **Prod API:** https://trustbot-api.fly.dev
- **Prod Web:** https://web-626.vercel.app
